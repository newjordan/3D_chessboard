# Final 12 Mixed 2D Pieces

Canonical mixed-source 2D SVG chess piece set exported from `chess_svg_library`.

## Files

- `wK.svg`, `wQ.svg`, `wR.svg`, `wB.svg`, `wN.svg`, `wP.svg`
- `bK.svg`, `bQ.svg`, `bR.svg`, `bB.svg`, `bN.svg`, `bP.svg`
- `final_selection.json` records the saved picker selection.
- `manifest.json` records the exported `library/final_12` set.

## Sources

- King, queen, rook, bishop: `lila` / `cooke`
- Knights: `deverac-piece-packager` / `kosal`
- White pawn: `deverac-piece-packager` / `alfarishy`
- Black pawn: `deverac-piece-packager` / `alfarishy-pink`
